# Feedback Hub

Статический сайт для хранения и публикации отчётов плейтестов.

## Структура

```
Feedback/
├── index.html              # Главная — список статей
├── css/styles.css          # Стили
├── js/main.js              # Загрузка списка статей
├── data/articles.json      # Метаданные статей
└── articles/
    └── 2026-07-07-playtest-raider.html
```

## Блоки отчёта

Каждая статья разбита на секции:

1. **Организация** — цели теста, регламент, шаблоны фидбека
2. **Геймплей** — механика стрельбы, отдача, ощущения
3. **Статы** — баланс оружия, кровотечение
4. **Итог** — выводы дня

## Добавление новой статьи

1. Создайте `articles/YYYY-MM-DD-slug.html` по образцу существующей статьи
2. Добавьте запись в `data/articles.json`
3. Закоммитьте и запушьте в `main`

## GitHub Pages

Сайт публикуется через GitHub Pages из ветки `main`, папка `/` (root).
